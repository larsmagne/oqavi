Open ¿Qué? And Value Infrastructure

Store and retrieve key/values pairs without any setup

Pricing

Using Oqavi is free. There is no setup. There are no API keys.
↑

Security

There is no security. There is no encryption. Anybody can probably snoop on your data. Anybody can overwrite your data if they can guess the keys. The NSA can definitely snoop on your data. My cat is probably reading your data as we speak.
↑

API

Oqavi returns JSONp data. Oqavi is not RESTful -- you use GET or POST as you want. Oqavi URLs are structured by slashes, where the elements are:

  • set/get
  • The JSONp function name to be returned
  • The "domain", which is vaguely your application name, or whatever else you choose, but can be whatever you want
  • The key
  • The value


↑

Examples

To store something in Oqavi, use the following URL: 

http://api.oqavi.no/set/mySetCallback/foo.org/foo/bar

To get something from Oqavi, use the following URL: 

http://api.oqavi.no/get/myGetCallback/foo.org/foo

The returned data will look like: 

myGetCallback({value: "bar"});

↑

Rate-limiting

There is no rate-limiting, although if you use it a lot, the service will probably get slower. The size of the key/value pairs are probably arbitrarily limited, so don't try to store images here. Any data you store here will probably get deleted after a while. Or not.
